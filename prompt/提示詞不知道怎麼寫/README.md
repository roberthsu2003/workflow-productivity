# 提示詞不知道怎麼寫？

面對空白輸入框，不知道該如何開口，或擔心寫得不夠準確？以下五種方式可以幫助你快速上手。

---

## 方式一：請 AI 直接提供 Prompt 範本

把你的目標告訴 AI，請它幫你產出一個可用的 prompt 範本，再依需求微調即可。

### 範例情境

**主題背景**：你是**華梵大學採購處**承辦人，收到「**OO 科技有限公司**」的設備報價，因本次預算有限無法合作，希望保留未來合作可能性。想請 AI 寫一封婉拒信，但不知道該怎麼下指令。

**你可以這樣問：**

> 我想請 AI 幫我寫一封婉拒廠商報價的信，語氣要禮貌但明確。請給我一個完整的 prompt 範本，讓我可以直接複製使用。  
> 請使用RTCCF(Role,Task,Context,Constraint,Format)的markdown格式

<details>
<summary>💡 參考解答（點擊展開）</summary>

<br>

AI 可能會回覆類似這樣的範本：
```
# 角色
你是一位有經驗的採購人員，擅長以得體的方式處理廠商往來信件。

## 任務
請幫我撰寫一封婉拒廠商報價的信件。

### 背景說明（請填入）
- 廠商名稱：
- 拒絕原因：
- 是否希望保留未來合作可能性：是 / 否

## 輸出格式
- 語氣：禮貌、正式、不拖泥帶水
- 結構：感謝來信 → 說明原因 → 表達未來合作意願（若適用）→ 署名
```

只要填入「背景說明」的欄位，再貼給 AI 即可使用。

</details>

## 方式二：請 AI 用一問一答的方式引導你

讓 AI 主動問你幾個問題，透過問答釐清需求，最後產出適合的 prompt。

### 範例情境

**主題背景**：你是大學生，下週課堂或活動要**輪流自我介紹**，想用 AI 幫你寫一段**可直接念出來**的稿子，但**不知道 prompt 要寫哪些條件才清楚**。這個情境**不需要任何檔案或長篇素材**：只要依 AI 問你的幾題在腦中想好（或隨手打幾個字），就能很快得到可用的 prompt，再貼一次就能得到成稿。

**你可以這樣問：**

> 我要準備一段課堂用的自我介紹，但不知道 prompt 怎麼寫。請用一問一答引導我，問我幾個關鍵問題，再依我的回答幫我產出完整的 prompt。我不需要上傳檔案，只要回答你的問題就好。  
> 請使用RTCCF(Role,Task,Context,Constraint,Format)的markdown格式

<details>
<summary>💡 參考解答（點擊展開）</summary>

<br>

AI 可能會先問你：

1. **使用場景**：一般課堂點名、社團破冰、還是線上會議？
2. **語言**：中文、英文，還是雙語各一段？
3. **長度**：約幾句話、約幾秒鐘內念完，或字數上限？
4. **一定要提到什麼**：科系、年級、興趣、想加強的能力、家鄉等（選幾項即可）？
5. **語氣**：輕鬆、正式、或帶一點幽默？

根據你的回答，AI 會組合出類似這樣的 **RTCCF（Markdown）** prompt（你再把「關於我的事實」填進 `Context` 或另開一則貼上即可）：
```markdown
## Role

你是協助撰寫口語自我介紹的助手，擅長把零碎資訊寫成自然、好念的開場白。

## Task

依使用者提供的事實，寫成一段**可直接口說**的開場自我介紹。

## Context

- 使用場景：（依問答填入，例如：大學課堂第一堂）
- 語言：（中文／英文）
- 長度：（例如：約 30 秒、或 120 字內）
- 語氣：（例如：輕鬆、不冗長）
- 必須涵蓋的資訊：（例如：姓名或暱稱、科系、一個興趣）

使用者會在下一則訊息提供關於自己的關鍵事實（科系、興趣等）；請僅依該內容撰寫。

## Constraints

- 不要捏造使用者未提供的事實。
- 若資訊不足，請在輸出開頭用一句話提醒還缺哪一項。

## Format

純文字**一段**，不要使用項目符號，方便直接念出。
```

</details>

## 方式三：直接搜尋現成的 Prompt 庫

網路上有許多分類完整的 prompt 庫，可以直接搜尋、複製使用，省去從零開始的時間。

### 常用 Prompt 庫資源

| 資源 | 說明 | 連結 |
|------|------|------|
| **第五 AI 提示詞** | 10 萬+ 提示詞，涵蓋寫作、商業、開發等分類，一鍵複製 | [prompt.diwuai.com](https://prompt.diwuai.com/prompts) |
| **Prompt Library** | 開源專案，支援中文，可依分類瀏覽與搜尋 | [mrxie23.github.io/PromptLibrary](https://mrxie23.github.io/PromptLibrary/) |
| **Studio Global** | 100+ 專家製作提示，支援繁體中文，可團隊共享 | [studioglobal.ai](https://www.studioglobal.ai/zh-hant/prompt-library/) |
| **PromptVault** | 500+ 模板，適合開發者、內容創作者、小型企業 | 直接搜尋「PromptVault」 |

### 使用步驟

1. 打開任一 prompt 庫，在搜尋欄輸入關鍵字（如「寫作」、「文章」）
2. 瀏覽結果，點選符合需求的 prompt
3. 一鍵複製，貼到 AI 對話框試用
4. 依實際需求微調（例如調整語氣、加入字數限制）

### 範例情境

**主題背景**：你是**華梵大學教務處**承辦人，有一份「**選課辦法說明**」寫得太冗長，想請 AI 改寫成更簡潔、適合大一新生閱讀的版本，字數控制在原文 80% 以內。

**從庫中複製或請 AI 協助時，可加上：**

> 請使用RTCCF(Role,Task,Context,Constraint,Format)的markdown格式

<details>
<summary>💡 從庫中找到的「文章改寫」prompt 範例（點擊展開）</summary>

<br>

```
請幫我改寫以下文章，要求：
1. 保持原意不變
2. 使用更簡潔、易懂的用詞
3. 適合 [目標讀者，例如：一般大眾 / 專業人士] 閱讀
4. 字數控制在原文的 80% 以內

原文：
（在此貼上你的文章）
```

把「目標讀者」和「原文」替換成你的內容即可使用。

</details>

## 方式四：從一句簡單需求開始，請 AI 幫你擴充

先寫出最簡單的一句需求，再請 AI 幫你補齊細節和結構，逐步完善。

### 範例情境

**主題背景**：你是**華梵大學資訊處**資深員工，下週有新人報到，想請 AI 寫「**資訊處新人到職注意事項**」，但腦中只有「幫我寫新人注意事項」這幾個字。

**你可以這樣問：**

> 我想請 AI 幫我寫一份新人到職注意事項，但我只想到「幫我寫新人注意事項」。請幫我把這句話擴充成一個完整、結構清楚的 prompt，包含角色、任務、需要涵蓋的內容與輸出格式。  
> 請使用RTCCF(Role,Task,Context,Constraint,Format)的markdown格式

<details>
<summary>💡 參考解答（點擊展開）</summary>

<br>

AI 可能會產出：
```
# 角色
你是一位熟悉公司流程的資深員工，負責協助新人順利融入。

## 任務
請幫我撰寫一份新人到職的注意事項清單。

### 需要涵蓋的內容
- 第一天報到的流程（時間、地點、聯絡人）
- 需要申請的系統帳號（Email、內部系統、差勤系統等）
- 部門日常工作的基本 SOP
- 常見問題與對應聯絡窗口

## 輸出格式
- 語氣：親切、易讀，讓新人不感到緊張
- 結構：以分段標題區分各主題，每個步驟用數字條列
- 若某項需依公司規定填寫，請標註「請依貴公司規定填寫」
```

之後可以依公司實際狀況，在「需要涵蓋的內容」中增減項目。

</details>

## 方式五：參考類似範例，套用結構改寫

找到「目標相近」的 prompt 範例，保留其結構框架，把內容替換成你需要的版本。

### 範例情境

**主題背景**：你是**華梵大學財務處**行政人員，需要製作「**出差費用申報說明**」供全校同仁參考。手邊有「會議通知信」的 prompt 可以參考，結構相近（角色、任務、需涵蓋內容、輸出格式）。

**你可以這樣做：**

1. 打開會議通知信的 prompt 範例
2. 保留原有的結構（角色、任務、需涵蓋的內容、輸出格式）
3. 把「會議通知」相關的文字，替換成「出差費用申報」的內容

**若請 AI 依此框架產出或檢查改寫稿，可加上：**

> 請使用RTCCF(Role,Task,Context,Constraint,Format)的markdown格式

<details>
<summary>💡 改寫後的 prompt 範例（點擊展開）</summary>

<br>

```markdown
# 角色
你是公司財務部門的行政人員，負責說明費用申報規定。

## 任務
請幫我製作一份出差費用申報的說明文件。

### 需要涵蓋的內容
- 可報銷的費用項目（交通、住宿、餐費等）
- 各項目的金額上限
- 需要檢附的單據種類
- 申報截止日期與送件方式

## 輸出格式
- 語氣：正式、清楚
- 結構：費用上限以表格呈現，其餘使用條列式說明
```

**改寫重點**：角色、任務、結構框架不變，只替換成「出差費用」相關的具體內容。

</details>

## 快速選擇指南

| 你的狀況 | 建議方式 |
|----------|----------|
| 完全不知道從何開始 | ✅ 方式二：請 AI 一問一答引導 |
| 有目標但不會結構化 | ✅ 方式一：請 AI 給範本，或方式四：從簡單句擴充 |
| 想快速套用現成模板 | ✅ 方式三：搜尋 prompt 庫 |
| 手邊有類似範例可參考 | ✅ 方式五：參考同類型改寫 |

> 💡 **小提醒**：無論使用哪種方式，產出的 prompt 都建議先試用一兩次，再根據實際效果微調。用得越多，就會越來越順手！
