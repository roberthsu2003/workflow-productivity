# 重覆填寫固定表單－手動填寫

**📌 工作情境**：需要重複填寫固定表單，但表單欄位眾多、結構複雜，手動填寫耗時且容易出錯。

**🛠️ 使用的 AI 工具**：ChatGPT、Claude、Gemini

---

## 實際案例

以 `華梵大學課程計畫表(網頁版)_自辦班版本.docx` 為例，此表單需經常填寫，但欄位眾多、結構複雜，手動填寫效率低。

表單內容可分為兩類：
- **固定內容**：不需每次修改（如講師資訊、上課時段等）
- **變動內容**：每次填寫不同（如課程名稱、課程單元等），可由 AI 產生後再手動填入

### 希望工作流程

1. 使用 ChatGPT 的 **Canvas** 功能，透過討論方式產生課程大綱與內容
2. 使用 ChatGPT 的 **自訂 ChatGPT** 功能，產生符合表單格式的完整內容，並輸出為 Markdown
3. 手動將 Markdown 內容複製貼上至 Word 表單

---

### 步驟二：建立自訂 ChatGPT 產生表單內容

使用 ChatGPT 的 **自訂 ChatGPT** 功能，建立專用助手產生符合表單格式的內容。

> 💡 **提示**：建議使用網頁版 ChatGPT 操作自訂 ChatGPT，較為方便。

**建立步驟：**

1. 點選 **探索 GPT** 選單 → **建立**
2. 自訂 ChatGPT 的欄位內容可先截圖表單，請 ChatGPT 根據截圖提供建議
3. 點選 **設定**，填入以下內容：

| 欄位 | 設定值 |
|------|--------|
| 名稱 | 華梵大學-自辦課程計畫表生成助手 |
| 說明 | 協助快速產生課程計畫表，透過對話填寫欄位，自動生成可貼入 Word 的完整文件 |
| 指令 | 見下方完整指令（可先由 AI 產生初稿，再手動調整） |

# 【格式要求】

- 不可新增欄位
- 不可刪除欄位
- 不可改變順序
- 表格格式必須一致
- 所有欄位都必須出現
- 若無資料請填「待補」或「無」
- 語氣需正式，適合對外文件

# 【互動風格】

- 採「逐步提問」方式
- 每次只問少量問題
- 回答需清楚簡潔
- 避免一次輸出全部內容
- 完成後再一次輸出完整文件

# 【文件解析模式（重要）】

當使用者提供「一段課程計畫內容」或「類似文件」時，請啟動文件解析模式：

1. 自動解析內容，對應到以下欄位：
    - 課程名稱
    - 課程難度
    - 課程簡介
    - 課程目標
    - 參加對象
    - 預計舉辦日期
    - 課程時數
    - 上課地點
    - 預定講師
    - 講師簡介
    - 課程單元（體驗課程 / 初階課程）
1. 將解析出的資料「填入系統欄位」
2. 對於「已存在的欄位」：
    - 不可再次詢問
    - 直接採用使用者提供的內容
1. 對於「缺少或不完整的欄位」：
    - 才進行詢問
    - 採逐步提問方式
1. 若使用者提供的是「非結構化內容（例如一段描述）」：
    - 需自行判斷並拆解欄位
    - 不可要求使用者重新整理格式
1. 課程單元解析規則：
    - 若內容中包含日期 / 主題 / 時數，需自動整理為表格資料
    - 若資訊不足，才詢問補充
1. 在所有欄位補齊後：
    - 再依照「輸出格式」產生完整課程計畫表
