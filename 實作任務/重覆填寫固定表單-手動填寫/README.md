# 重覆填寫固定表單-手動填寫

**📌 工作情境**：需要重複填寫固定表單，但表單欄位眾多、結構複雜，手動填寫耗時且容易出錯。

**🛠️ 使用的 AI 工具**：ChatGPT、Claude、Gemini

---

## 實際案例

以 `華梵大學課程計畫表(網頁版)_自辦班版本.docx` 為例，此表單需經常填寫，但欄位眾多、結構複雜，手動填寫效率低。

表單內容可分為兩類：
- **固定內容**：不需每次修改（如講師資訊、上課時段等）
- **變動內容**：每次填寫不同（如課程名稱、課程單元等），可由 AI 產生後再手動填入

### 希望工作流程

**方式 A：從零開始（需先產生課程大綱）**

1. 使用 ChatGPT 的 **Canvas** 功能，透過討論方式產生課程大綱與內容
2. 建立 **自訂 ChatGPT**（課程計畫表生成助手），上傳表單至知識庫
3. 啟動對話，將 Canvas 產出的 Markdown 課程內容貼入，或由 AI 逐步詢問欄位
4. AI 依指令產生符合表單格式的完整內容（含固定欄位、自動生成欄位），輸出為 Markdown
5. 手動將 Markdown 內容複製貼上至 Word 表單

**方式 B：已有課程計畫內容（文件解析模式）**

1. 建立 **自訂 ChatGPT**（課程計畫表生成助手），上傳表單至知識庫
2. 直接貼入「一段課程計畫內容」或「類似文件」
3. AI 自動解析內容、對應欄位，對缺少的欄位採逐步提問補齊
4. AI 產出完整課程計畫表（Markdown 格式）
5. 手動將內容複製貼上至 Word 表單
