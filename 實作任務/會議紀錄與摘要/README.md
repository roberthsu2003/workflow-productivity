# 任務 1：撰寫會議紀錄與摘要

## 🎯 工作情境

每週團隊會議後，需要將 60 分鐘的會議內容整理成結構化的紀錄，並提取關鍵決議事項。

## 🤖 使用的 AI 工具

- **NotebookLM**（Google Meet 錄音/逐字稿上傳）
- **Claude** 或 **ChatGPT**（結構化整理）
- **Google Docs**（協作編輯）

## 💻 AI 負責的工作（80%）

- 自動轉錄語音成文字
- 識別並提取討論主題
- 整理發言重點
- 生成會議結構（討論內容、決議事項、行動項目）
- 格式化文件

## 👤 你負責的工作（20%）

- 確認會議錄音/逐字稿上傳
- 檢視並修正 AI 提取的關鍵點
- 確認決議事項的準確性
- 標記負責人和截止日期
- 最終審核並發送

---

## 📋 範例說明

### 範例來源

[衛生福利部中央健保署_全民健康保險藥物給付項目及支付標準共同擬訂會議特材部分第75次(114年3月)會議紀錄及附錄](https://www.nhi.gov.tw/ch/cp-18105-49ecc-3930-1.html)

### 範例主題

**肺動脈高壓處置卡匣健保支付點數調整案**

- 來源：21 個會議記錄錄音檔

### 目標

1. 將 21 個會議記錄錄音檔轉換為逐字稿
2. 自動建立重點摘要與決議事項
3. 手動建立重點摘要與決議事項

---

## 📝 實作步驟詳解

### 步驟 1：將錄音檔轉換為逐字稿

**AI 工具**：NotebookLM

**使用 prompt**：

```
我需要將所有的 mp3 檔，建立一個逐字稿的 markdown 檔案。
輸出格式範例：

[01主席致詞.mp3]
內容: xxxxx
重要內容: xxxxx
-----
[02參閱並確認上次會議紀錄.mp3]
內容: xxxxx
重要內容: xxxxx
```

### 步驟 2：自動建立重點摘要與決議事項

**AI 工具**：NotebookLM **報告按鈕功能**

> 💡 **注意**：報告按鈕使用自動產生的 prompt，可手動產生更適合的 prompt 以優化結果。

### 步驟 3：手動建立重點摘要與決議事項

**AI 工具**：NotebookLM **自訂 prompt 功能**

```
撰寫一份綜合性簡報，提煉資料來源的主要主題和觀點。
目的: 建立簡捷的重點摘要與決議事項
輸出格式: markdown 檔案
輸出內容:
- 重點摘要
- 決議事項
- 行動事項
- 其他事項
```

### 步驟 4：產生會議重點簡報內容

**AI 工具**：NotebookLM **prompt 功能**

```
產生一份會議重點簡報內容
```

### 步驟 5：自動產生會議重點簡報

**AI 工具**：NotebookLM **簡報功能**

## 📚 NotebookLM 背景知識

### 一、為什麼公務機關需要 NotebookLM

**政府單位常見情境**：

- 會議錄音整理
- 訪談紀錄彙整
- 長會議逐字稿
- 政策討論紀錄
- 跨單位會議
- 公文重點摘錄

**痛點**：

- 逐字稿很長難閱讀
- 重點難抓
- 行動事項容易漏
- 跨會議難比較
- 彙整耗時

**NotebookLM 可協助**：

- 自動摘要會議
- 萃取決議事項
- 整理行動項目
- 產生會議紀錄草稿
- 比較多場會議
- 生成訪談摘要

👉 適合公部門文件型工作

---

### 二、NotebookLM 在會議與訪談的角色

**NotebookLM = 會議與訪談分析助理**

| 輸入 | 輸出 |
|------|------|
| 逐字稿 | 會議紀錄 |
| 錄音稿 | 重點摘要 |
| 訪談 | 主題整理 |
| 多場會議 | 綜合結論 |
| 內容 | 行動事項 |

---

### 三、使用時機（公務情境）

| 情境 | 上傳 | 產生 |
|------|------|------|
| **會議後整理紀錄** | 逐字稿、錄音轉文字 | 會議摘要、決議事項、待辦事項 |
| **訪談紀錄整理** | 訪談逐字稿 | 訪談重點、主題分類、需求整理 |
| **多場會議彙整** | 多份會議紀錄 | 共通議題、差異、決策演進 |
| **公文或簡報素材** | 會議紀錄 | 報告重點、簡報大綱、政策摘要 |

### Demo 步驟

| 步驟 | 操作 | 展示結果 |
|------|------|----------|
| **1. 上傳逐字稿** | 上傳勞動政策會議逐字稿.txt 到 NotebookLM | 內容含：就業方案討論、補助措施、執行困難、決議事項 |

| **3. 萃取決議事項** | 問：「請列出本次會議決議事項」 | 決議 1、決議 2、決議 3 👉 公務員最有感 |
