### 免費方案使用技巧
1. **分散使用**：不同工具用於不同場景，避免單一工具額度耗盡
2. **批次處理**：累積多個任務一次處理，提高效率
3. **本地優先**：敏感資料使用 Ollama，一般資料用雲端工具
4. **善用額度**：定期檢查各工具的免費額度使用狀況

### 資料安全提醒
- ⚠️ **敏感資料**：使用 Ollama 本地處理，不上傳雲端
- ✅ **一般資料**：可使用 ChatGPT、Claude、Gemini 免費版
