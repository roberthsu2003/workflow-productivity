## 學生複製區（純文字，可直接複製）

> **小提示：** 在 GitHub／VS Code 預覽中，點程式碼區塊**右上角**常可一鍵複製；或於區塊內三擊滑鼠選取整段文字。

### 複製 1：NotebookLM 產出簡報 Prompt

```markdown
請依下列規範產出本簡報內容，不更改原始輸入內容：

- 科技感與專業風格，適合高階主管簡報
- 字體全程統一（如思源黑體/標楷體/無襯線，不混用其他字型）
- 每張投影片重點不超過 3 項
- 需附上配色建議（主色、輔色、背景色、字色 HEX），例如：
    - 主色：#20B2AA（藍綠）
    - 輔色：#00FFD0（螢光青綠）
    - 背景色：#1A1D23（深灰黑）
    - 字色：#FFFFFF（白）、#CCCCCC（淺灰）
- 插圖建議：「線條圖示」、「抽象科技感背景」，避免過度複雜或照片
- 內容僅產出重點，文字精簡、不改動原意
```

### 複製 3：簡報大綱 Prompt（ROSES 框架）

```markdown
## R – 角色設定
你是一位擅長科技主題簡報的簡報顧問，熟悉數位轉型、AI 應用等議題。

## O – 任務目標
幫我產出一份「2025 校園數位轉型計畫」的簡報大綱，約 8 頁，風格偏科技感。

## S – 執行步驟
1. 封面頁（標題、副標、報告單位、日期）
2. 6～8 張投影片，每張有標題與 3 個重點

3. 結尾頁（總結或 Q&A）

## E – 範例格式
第 1 張：封面
第 2 張：校園數位轉型現況
  - 重點 1、重點 2、重點 3
（以此類推）

## S – 範圍與風格
- 語言：繁體中文
- 風格：科技感、專業、適合向高階主管簡報
- 每張投影片重點不超過 3 項
```

## 主軸教學：NotebookLM 產出與 PPT 模板套用

本段為雙主線整理：**① NotebookLM 產出簡報**、**② PPT 模板取得與套入 PowerPoint**（含與 NotebookLM 的搭配）。**Prompt 全文請用上方「學生複製區」**，此處不重複貼上。

**建議總流程：** 先**大綱**（可選）→ **來源文件 + NotebookLM 產出** → **對照 PPT 模板**調整版型／匯出 → 校稿與演練。

### 一、NotebookLM 產出簡報

#### 1.1 適合什麼情境

| 情境 | 說明 |
|------|------|
| 手邊已有 **PDF／Google 文件／長篇計畫** | NotebookLM 以**來源為準**整理章節，較能對齊原文、降低空談 |
| 要**快速得到簡報初稿** | 一鍵或幾步產出簡報型輸出 |
| 教學、校內簡報、內部報告 | 搭配來源出處較好交代依據 |

#### 1.2 操作步驟

1. 建立 Notebook，**上傳**與主題相關的 PDF、Google 文件等**來源**。
2. 在 NotebookLM 中選擇產生 **簡報**（或當前版本提供的簡報／簡報風格輸出；介面名稱可能更新）。
3. 檢視章節是否對應你的文件；可再請它「**每頁重點濃縮至 3 行以內**」「**不要延伸未出現在來源的數據**」。
4. **產出物**：多為**簡報圖像或簡報型版面**，依版本可能**不易直接編輯文字**；新版本若支援以自然語言調整內容，可優先使用官方說明中的編輯方式。

#### 1.3 優點與限制

| 優點 | 限制 |
|------|------|
| 依**上傳來源**整理，較貼近既有文件 | 產出常為**圖片為主**，若需可編輯 `.pptx` 文字，往往要**另套模板或重打**（見第二節） |
| 省下手動從長文摘重點的時間 | 版型受工具限制，**品牌模板**需到 PowerPoint 再對齊 |

**完成檔參考：** [NotebookLM 簡報產出範例](./簡報完成檔案)

> **提示**：NotebookLM 功能更新頻繁，按鈕名稱以 [官方說明](https://support.google.com/notebooklm) 與當前介面為準。

### 二、PPT 模板產出與套用（PowerPoint）

#### 2.1 為什麼要「模板」

- **視覺一致**：封面、內頁、標題層級、色票統一，像同一套簡報。
- **省時間**：不用從白投影片開始拉線條與對齊。
- **可編輯**：下載的 `.pptx` 在 **Microsoft PowerPoint**（或相容軟體）裡**逐頁改字、改圖**，符合交付需求。

#### 2.2 模板哪裡找（參考來源）

- [Slidesgo](https://slidesgo.com/)
- [Slidebean](https://slidebean.com/)
- [Slides Carnival](https://www.slidescarnival.com/)
- 靈感參考（注意版權）：[Pinterest](https://www.pinterest.com/)、[Behance](https://www.behance.net/)

挑選時先決定：**深／淺色底**、**是否含圖表頁**、**頁數是否夠用**，再下載 PowerPoint 格式。

#### 2.3 設計元素與版權提醒

若從網路模板**截圖**當「靈感」：

- 請**不要直接抄襲**有版權的完整版面作為商業重製。
- 可記錄**配色 HEX、區塊比例、字級層級**，在自己的模板或 NotebookLM 提示中**重新描述**。

**本專案範例圖（封面／內頁參考）：** [簡報範例元素 source](./簡報範例元素source/)

#### 2.4 把 NotebookLM 與 PPT 模板串在一起（推薦工作流）

**流程：**

1. **選定並下載**一套 `.pptx` 模板（封面 + 章節 + 內容頁版型）。
2. 將**計畫書 PDF**（與需要時的**設計參考圖**）上傳 NotebookLM，產出簡報（搭配用 Prompt 見**複製 2**）。
3. 在 **PowerPoint** 中**開啟模板**，將 NotebookLM 各頁的**標題與三點重點**複製到對應版型；若 NotebookLM 給的是整張圖，可**插入為圖**或**只抄文字**到模板文字框（依清晰度與編輯需求決定）。
4. 統一字體與色票與模板一致（例如全用思源黑體）。

若介面中**沒有**「簡報」按鈕，可先請 NotebookLM 產出**條列式綱要（每頁標題 + 三點）**，再**手動貼入** PowerPoint 模板。

### 三、先確定大綱（可選，但建議）

沒有長文件、只有主題時，可先用 AI 產**大綱**（Prompt 見**複製 3**），再餵給 NotebookLM 或自己逐頁填入模板。

**主題背景範例**：你是**華梵大學資訊處**承辦人，需向校務會議簡報「**2025 校園數位轉型計畫**」，內容含 AI 工具導入、智慧教室建置、資安升級。希望產出約 **8 頁**簡報大綱，風格偏科技感。

**8 頁大綱參考範例**見上方**複製 6**。

### 四、風格與配色（簡要）

| 項目 | 建議 |
|------|------|
| **配色** | 深色底 + 霓虹藍／青綠／電紫點綴；或白底 + 單一科技主色（如 #0066CC） |
| **字體** | 標題無襯線粗體（如思源黑體 Heavy）；內文易讀、少混用字體 |
| **原則** | 一頁一重點（One Slide, One Idea）；對比清楚 |

自然語言範例見**複製 5**。

### 五、選讀：其他產出方式（摘要）

詳見下方**附錄**「使用 AI 工具」各方法。重點：Gemini／ChatGPT／Gamma／Canva 等可與「大綱 → 模板」流程並用。

### 六、需求快速對照（主軸）

| 你的需求 | 建議 |
|----------|------|
| 有長 PDF／報告，要快出簡報初稿 | **NotebookLM**（複製 1） |
| 要可交給長官、版型專業、可改字 | **PPT 模板** + 將 NotebookLM 文字／圖套入 |
| 沒有文件、只有主題 | **大綱**（複製 3）→ 寫成短文件或逐頁手打，再視需要餵 NotebookLM |
| 一定要 `.pptx` 且版面精美 | **模板下載**為主，NotebookLM 為內容來源 |

### 附錄 C｜手動工具（複製貼上）

- 內建 AI 輔助工具（依軟體版本而異）

**常見工具：**

- Microsoft PowerPoint  
- Mac Keynote  
- Google Slides  
- Canva  
- Gamma  

**優點：**

- 操作直覺、自由度高，易於細緻調整版面與設計。
- 熟悉度高，大部分人已會使用。
- 可以快速套用自訂範本與設計元素。
- 適合進階使用者調整格式、動畫、轉場等細節。

**缺點：**

- 需要人工逐頁輸入與排版，流程較耗時。
- 難以自動總結長文資料、產生結構。
- 圖片、數據整合需自行搜尋與插入，比較費力。
- 不易快速產生初稿、缺少 AI 一鍵產生的便利。

#### 附錄 D3｜方法 3：使用 Gamma 生成簡報

1. 前往 [Gamma](https://gamma.app)（需註冊），選擇「用 AI 建立」。

3. Gamma 會自動排版與分頁；可再調整主題樣式、替換圖片。

- **適合**：要快速得到**視覺完整**的初版，再微調內容與品牌。

**截圖並去除文字，取得設計元素**

可從模板截圖背景與版型，作為「視覺參考」上傳至 NotebookLM 或給其他 AI，請其**不要抄襲版權圖**，僅參考配色與區塊配置。

[**簡報設計元素範例（圖檔）**](./簡報範例元素source/) — 內含封面與內頁參考圖。

**利用 NotebookLM 產生簡報：** 上傳計畫書 PDF 與（選用）設計參考圖後，Prompt 見**複製 2**。

> 💡 **提示**：NotebookLM 實際介面可能隨版本更新，若無「簡報」按鈕，可先產生**條列式綱要**，再手動貼至 PowerPoint 並套用模板。
