## 使用方式

1. 在支援 **RAG / 知識庫** 的工具（如 ChatGPT 專案、Claude Projects、Gemini 的檔案參考、NotebookLM 等）中，依平台指示**上傳**本資料夾內檔案（可只上傳根目錄、只上傳 `短的pdf`、或混合）。
2. 提問時請**限定依知識庫回答**，並觀察模型是否**引用**內容、是否出現**幻覺**。
3. 可任選一檔或多檔合併，比較「僅文字 `.txt`」、**長篇 PDF** 與 **短篇產品 PDF** 在檢索與回答品質上的差異。
4. 驗證準確度時，建議優先使用 **`短的pdf`** 三檔搭配主教學中的**示範問答**，較易對照答案。
