# 知識庫原始檔

本資料夾提供 **RAG（檢索增強生成）** 教學與實作時可上傳至 AI 工具「知識庫／專案檔案」的**範例素材**。內容為**教學虛構或改寫之公開領域類型文件**，不應視為真實金融、醫療或產品官方文件；實務專案請改用經授權或自有資料。

## 檔案一覽

| 檔案 | 格式 | 內容概要 | 教學用途提示 |
|------|------|----------|
| [信用卡權益說明.txt](./信用卡權益說明.txt) | 純文字 | 虛構信用卡（卡別、年費、回饋、權益條款等）之長篇說明 | 適合練習**文字切割**、**條款問答**、**數字與條件檢索** |
| [智慧型手機使用手冊.pdf](./智慧型手機使用手冊.pdf) | PDF | 智慧型手機類**使用手冊**範例 | 適合練習**PDF 上傳**、**圖文混排文件**之檢索 |
| [電動機車使用手冊.pdf](./電動機車使用手冊.pdf) | PDF | 電動機車類**使用手冊**範例 | 同上；可與手機手冊對照**多主題知識庫** |
| [健保就醫指南.pdf](./健保就醫指南.pdf) | PDF | 就醫流程／指南類**參考文件**範例 | 適合練習**分類導覽式問答**（實際醫療決策仍以官方與醫師為準） |

## 使用方式

1. 在支援 **RAG / 知識庫** 的工具（如 ChatGPT 專案、Claude Projects、Gemini 的檔案參考等）中，依平台指示**上傳**本資料夾內檔案。
2. 提問時請**限定依知識庫回答**，並觀察模型是否**引用**內容、是否出現**幻覺**。
3. 可任選一檔或多檔合併，比較「僅文字 `.txt`」與「**PDF**」在檢索與回答品質上的差異。
