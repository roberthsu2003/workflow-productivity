> 3份簡單的產品說明  
> 無線耳機,咖啡機,室內植物土壤濕度和光照監測器

1. **數值與規格檢索題**

```other
SonicCloud 9 耳機在開啟與關閉主動降噪 (ANC) 模式下，電池續航力分別是多少？
```

答案： 開啟 ANC 時續航力為 30 小時 ，關閉 ANC 時續航力為 45 小時 。

```other
BrewBot X1 咖啡機的水箱容量是多少？它支援哪種頻率的 WiFi？
```

答案： 水箱容量為 1.5 公升 。它僅支援 2.4GHz WiFi，不支援 5GHz 。

2. **操作程序題**

```other
如何將 BrewBot X1 咖啡機恢復出廠設定？
```

答案： 需要同時長按「電源」按鈕和「醸造」按鈕 10 秒 。

```other
SonicCloud 9 與 FloraGard Pro 上的閃爍紅燈分別代表什麼意思？
```

答案：耳機是低電量；植物監測器是水分嚴重不足。

```other
在使用 SonicCloud 9 耳機時，如何透過觸控板切換到「通透模式」？
```

答案： 在右耳罩觸控板上長按 2 秒，即可切換 ANC/通透模式 。



3. **狀態燈號判讀題**

```other
FloraGard Pro 的 LED 燈閃爍紅色代表什麼意思？使用者該如何處理？
```

答案:紅色閃爍代表濕度過低 。使用者請立即澆水 。


```other
當 BrewBot X1 的「清潔」指示燈變成橙色時，代表發生了什麼事？
```

答案： 這代表機器已經過了 300 次釀造循環，需要進行除垢保養 。



4. **故障排除與限制題**

```other
BrewBot X1 顯示錯誤代碼 "E03" 是什麼問題？該如何解決？
```

答案：E03 代表過熱保護觸發 。解決方法是拔掉電源等待 30 分鐘 。

```other
我可以把 FloraGard Pro 放在戶外淋雨嗎？
```

答案： 雖然可以在戶外使用，但它不是完全防水的，請勿暴露在大雨中 。

---

5. **安裝與維護細節題**

```other
安裝 FloraGard Pro 時，建議插入土壤的深度是多少？
```

答案： 建議將探針插入土壤中約 3 到 4 英吋深 。

```other
SonicCloud 9 耳機如果完全沒電了，使用 USB-C 充滿電需要多久時間？
```

答案： 充滿電需要 2 小時 。

#### 健保就醫指南（限制模型只能依據 RAG 文件回答、避免幻覺、強化準確度評估）

<details>
<summary>點擊展開：健保就醫指南 — 題目與參考答案</summary>

> 知識庫檔案：[健保就醫指南.pdf](./知識庫原始檔/健保就醫指南.pdf)  
> 以下為教學用示範題；實際費用、給付與申請規定以主管機關及保險人最新公告為準。

1. **費用與補發**

```other
若健保卡遺失，申請補發需要支付多少工本費？
```

答案： 申請補發健保卡需要準備 **200 元** 的工本費。

```other
在沒有轉診單的情況下，直接前往「醫學中心」門診的部分負擔費用是多少？
```

答案： 若無轉診單，前往醫學中心的部分負擔費用為 **550 元**。

2. **轉診與部分負擔**

```other
持有轉診單前往「區域醫院」就醫，部分負擔費用會從 420 元降為多少？
```

答案： 持轉診單至區域醫院，部分負擔費用將減免為 **240 元**。

```other
請列出至少五類可以免除門診部分負擔的對象。
```

答案： 免部分負擔對象包括：**重大傷病患者**、**分娩**、**山地離島地區就醫**、**3 歲以下兒童**、**低收入戶**、榮民及接受預防保健服務者。

3. **處方與急診**

```other
慢性病連續處方箋最多可以領幾次藥？第三次領藥前有何規定？
```

答案： 最多可**連續領藥 3 次**，且在**第 3 次領藥前必須回診**。

```other
急診檢傷分級中，哪幾個等級的患者可以免除部分負擔？
```

答案： 檢傷分級為 **1 至 2 級** 的患者免除部分負擔。

4. **住院與給付範圍**

```other
健保住院部分負擔的「全年累計」最高限額是多少？
```

答案： 全年累計的部分負擔上限為 **69,000 元**。

```other
哪些牙科項目是不屬於健保給付範圍，需要民眾自費的？
```

答案： 不給付項目包括：**植牙**、**齒列矯正**、**美白**、**貼片**、牙周病雷射治療。

---

5. **預防保健與海外就醫**

```other
40 歲至 64 歲的民眾，接受成人預防保健服務的頻率為何？
```

答案： 該年齡層民眾**每 3 年可接受 1 次**檢查。

```other
在國外突發疾病就醫，應於返國後多久期限內申請自墊醫療費用核退？
```

答案： 必須在 **6 個月內** 提出申請。

</details>

#### ChatGPT Projects：客服／諮詢 Prompt（RAG：健保就醫指南）

建立 Project 並上傳 [健保就醫指南.pdf](./知識庫原始檔/健保就醫指南.pdf) 後，將下方全文貼入 **Instructions**（或自訂 GPT 的**指令**），即可得到**簡短、有據**的客服風格回覆；若知識庫未涵蓋，模型應明確說明無法判斷，避免幻覺。

<details>
<summary>點擊展開：客服 Prompt 全文（請全選複製）</summary>

```text
## 角色與任務
你是**健保就醫諮詢**第一線客服。你只能依據本專案／對話中已提供之**《健保就醫指南》**（RAG 知識庫）內容回答，不得憑空補充法規細節或官方未在文件中出現的數字。

## 回應風格（務必簡潔）
- 使用**繁體中文**，語氣專業、有禮。
- **先給結論**（一句話或一個短段落），若使用者需要細節再補**最多 2～4 行**；避免長篇背景與重複敘述。
- 條列時每點**一行為限**，總點數盡量不超過 5 點。
- 涉及**金額、天數、次數、對象**時，須與文件一致；若文件有多種情境，先問清使用者情境再簡答。

## 知識邊界與拒答
- 若問題在知識庫中**找不到依據**，請只回覆類似：「依目前上傳之《健保就醫指南》內容無法回答此問題，請洽健保署或您的投保單位確認。」**不要猜測或編造**。
- 不提供醫療診斷、不保證個案是否給付；可簡短提醒「實際給付以現場與官方認定為準」。
- 本文件若為教學範例，必要時可附一句：「以上僅依參考文件說明，實務以主管機關最新公告為準。」

## 輸出格式建議
- 一般問答：**結論** →（選填）**補充 1～2 句**。
- 需要分項時：使用簡短條列，不加冗長前言與結語套話。
```

</details>
