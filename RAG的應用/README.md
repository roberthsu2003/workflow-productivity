<summary>點擊展開：產品說明 — 題目與參考答案</summary>

> 3份簡單的產品說明  
> 無線耳機,咖啡機,室內植物土壤濕度和光照監測器

1. **數值與規格檢索題**

```other
SonicCloud 9 耳機在開啟與關閉主動降噪 (ANC) 模式下，電池續航力分別是多少？
```

答案： 開啟 ANC 時續航力為 30 小時 ，關閉 ANC 時續航力為 45 小時 。

```other
BrewBot X1 咖啡機的水箱容量是多少？它支援哪種頻率的 WiFi？
```

答案： 水箱容量為 1.5 公升 。它僅支援 2.4GHz WiFi，不支援 5GHz 。

2. **操作程序題**

```other
如何將 BrewBot X1 咖啡機恢復出廠設定？
```

答案： 需要同時長按「電源」按鈕和「醸造」按鈕 10 秒 。

```other
SonicCloud 9 與 FloraGard Pro 上的閃爍紅燈分別代表什麼意思？
```

答案：耳機是低電量；植物監測器是水分嚴重不足。

```other
在使用 SonicCloud 9 耳機時，如何透過觸控板切換到「通透模式」？
```

答案： 在右耳罩觸控板上長按 2 秒，即可切換 ANC/通透模式 。



3. **狀態燈號判讀題**

```other
FloraGard Pro 的 LED 燈閃爍紅色代表什麼意思？使用者該如何處理？
```

答案:紅色閃爍代表濕度過低 。使用者請立即澆水 。


```other
當 BrewBot X1 的「清潔」指示燈變成橙色時，代表發生了什麼事？
```

答案： 這代表機器已經過了 300 次釀造循環，需要進行除垢保養 。



4. **故障排除與限制題**

```other
BrewBot X1 顯示錯誤代碼 "E03" 是什麼問題？該如何解決？
```

答案：E03 代表過熱保護觸發 。解決方法是拔掉電源等待 30 分鐘 。

```other
我可以把 FloraGard Pro 放在戶外淋雨嗎？
```

答案： 雖然可以在戶外使用，但它不是完全防水的，請勿暴露在大雨中 。

---

5. **安裝與維護細節題**

```other
安裝 FloraGard Pro 時，建議插入土壤的深度是多少？
```

答案： 建議將探針插入土壤中約 3 到 4 英吋深 。

```other
SonicCloud 9 耳機如果完全沒電了，使用 USB-C 充滿電需要多久時間？
```

答案： 充滿電需要 2 小時 。

#### ChatGPT Projects：客服／諮詢 Prompt（RAG：健保就醫指南）

建立 Project 並上傳 [健保就醫指南.pdf](./知識庫原始檔/健保就醫指南.pdf) 後，將下方全文貼入 **Instructions**（或自訂 GPT 的**指令**）。本 Prompt 採用本倉庫 [AI 提示詞工程指南](../prompt/AI提示詞工程指南/README.md) 的 **ROSES 框架**（角色、目標、步驟、輸出格式、範圍與風格）。

<details>
<summary>點擊展開：客服 Prompt 全文（ROSES；請全選複製）</summary>

```text
## R – 角色設定
你是**健保就醫諮詢**第一線客服，回答必須**僅依**本專案／對話中已提供之**《健保就醫指南》**（RAG 知識庫）內容；不得憑空補充法規細節，不得捏造官方未在文件中出現的數字或條文。

## O – 任務目標
在使用者詢問與健保就醫、給付、費用、申請流程等相關問題時，提供**簡潔、可讀、有依據**的說明；若知識庫未涵蓋該題，**明確表示無法依現有文件回答**，並引導洽官方或投保單位，避免幻覺。

## S – 執行步驟
1. **釐清意圖**：辨識使用者問的是費用、對象、流程、期限或給付範圍中的哪一類；若題意模糊，**先以一句話追問**必要條件（例如是否持轉診單、就醫層級），再作答。
2. **僅用知識庫**：只根據《健保就醫指南》中可對應之內容回答；不可加入文件未載明之推測。
3. **結論優先**：先給**一句核心結論**，再視需要補充**最多 2～4 行**；避免長篇背景與重複敘述。
4. **數字與條件**：涉及金額、天數、次數、對象時，須與文件一致；同一議題若有多種情境，應分項**簡短**說明或先釐清情境。
5. **無法回答時**：若檢索不到依據，只回覆：「依目前上傳之《健保就醫指南》內容無法回答此問題，請洽健保署或您的投保單位確認。」**不猜測、不編造**。
6. **非客服範圍**：不提供醫療診斷、不斷言個案是否一定給付；必要時一句話提醒「實際給付以現場與官方認定為準」。

## E – 輸出格式
- **一般問答**：先一行**結論**，下一行空行後（選填）**補充 1～2 句**；不加冗長開場白與結語套話。
- **需分項時**：使用 `-` 條列，**每點一行**，**總點數不超過 5 點**。
- **無法依文件回答時**：單獨一段，僅使用步驟 5 之固定說明，不補充猜測內容。
- **教學情境**（選用）：文末可另起一行附註：「以上僅依參考文件說明，實務以主管機關最新公告為準。」

## S – 範圍與風格
- 語言：**繁體中文**；語氣專業、有禮、**簡潔**（能一句說完勿用兩句）。
- 不提供醫療處置建議；不代替官方釋義或個案核定。
- 不引用知識庫以外的「據說／一般來說」等籠統說法。
```

</details>

#### 健保就醫指南（限制模型只能依據 RAG 文件回答、避免幻覺、強化準確度評估）

<details>
<summary>點擊展開：健保就醫指南 — 題目與參考答案</summary>

> 知識庫檔案：[健保就醫指南.pdf](./知識庫原始檔/健保就醫指南.pdf)  
> 以下為教學用示範題；實際費用、給付與申請規定以主管機關及保險人最新公告為準。

1. **費用與補發**

```other
若健保卡遺失，申請補發需要支付多少工本費？
```

答案： 申請補發健保卡需要準備 **200 元** 的工本費。

```other
在沒有轉診單的情況下，直接前往「醫學中心」門診的部分負擔費用是多少？
```

答案： 若無轉診單，前往醫學中心的部分負擔費用為 **550 元**。

2. **轉診與部分負擔**

```other
持有轉診單前往「區域醫院」就醫，部分負擔費用會從 420 元降為多少？
```

答案： 持轉診單至區域醫院，部分負擔費用將減免為 **240 元**。

```other
請列出至少五類可以免除門診部分負擔的對象。
```

答案： 免部分負擔對象包括：**重大傷病患者**、**分娩**、**山地離島地區就醫**、**3 歲以下兒童**、**低收入戶**、榮民及接受預防保健服務者。

3. **處方與急診**

```other
慢性病連續處方箋最多可以領幾次藥？第三次領藥前有何規定？
```

答案： 最多可**連續領藥 3 次**，且在**第 3 次領藥前必須回診**。

```other
急診檢傷分級中，哪幾個等級的患者可以免除部分負擔？
```

答案： 檢傷分級為 **1 至 2 級** 的患者免除部分負擔。

4. **住院與給付範圍**

```other
健保住院部分負擔的「全年累計」最高限額是多少？
```

答案： 全年累計的部分負擔上限為 **69,000 元**。

```other
哪些牙科項目是不屬於健保給付範圍，需要民眾自費的？
```

答案： 不給付項目包括：**植牙**、**齒列矯正**、**美白**、**貼片**、牙周病雷射治療。

---

5. **預防保健與海外就醫**

```other
40 歲至 64 歲的民眾，接受成人預防保健服務的頻率為何？
```

答案： 該年齡層民眾**每 3 年可接受 1 次**檢查。

```other
在國外突發疾病就醫，應於返國後多久期限內申請自墊醫療費用核退？
```

答案： 必須在 **6 個月內** 提出申請。

##### 《智慧型手機使用手冊》

```other
根據規格總覽，這款手機包裝內容物包含哪些東西？
```

答案： 手機主機、USB-C 充電線、**65W 充電器**、SIM 卡退卡針、**透明保護殼**、使用手冊及保固卡。

```other
使用原廠 65W 快速充電時，從 0% 充至 70% 需要多久？
```

答案： **30 分鐘**可充至 70%。

```other
後置主相機中，長焦鏡頭的畫素與光學變焦倍數為何？
```

答案： **800 萬畫素**、**3 倍光學變焦**。

```other
手冊建議將螢幕自動關閉時間設定在什麼區間以延長續航力？
```

答案： **30 秒至 1 分鐘**之間。

```other
設定指紋辨識時，使用者需要重複按壓感應區幾次？
```

答案： **15～20 次**。

```other
Wi-Fi 6 最高傳輸速度可達多少？
```

答案： **1.2Gbps**。

```other
觸控螢幕無反應時，進入「安全模式」測試的操作步驟是什麼？
```

答案： 關機後長按電源鍵，**看到 Logo 時改長按音量下鍵**。

```other
應用程式閃退時，手冊建議清理快取的路徑為何？
```

答案： **設定 > 應用程式 > 選擇應用程式 > 儲存空間 > 清除快取**。

```other
標準保固期限多久？列舉三項不在保固範圍內的情況。
```

答案： **1 年**；不含**人為損壞**、**進水**、**摔機**（依手冊）。

```other
進行系統更新前，電量應保持在多少百分比以上？
```

答案： **50% 以上**。

##### 《電動機車使用手冊》

```other
ES-3000 動力系統的最大扭力是多少？
```

答案： **26 N·m**。

```other
增程版（三電池）在市區騎乘的續航里程約為多少公里？
```

答案： 約 **120～135 公里**。

```other
單人騎乘時，後輪標準胎壓應設定為多少？
```

答案： **2.50 kg/cm²**。

```other
使用家用充電器為單顆電池充電（0～100%）需要多少時間？
```

答案： 單顆約 **3 小時**。

```other
新車「初次保養」時機與費用？
```

答案： 行駛 **1,000 公里或 1 個月**；費用**免費**。

```other
頻繁使用快速充電站，預計會加速電池老化約多少？
```

答案： 約 **15～20%**。

```other
儀表錯誤代碼「E002」代表什麼？如何處理？
```

答案： **馬達過熱保護**；停車待冷卻 **10～15 分鐘**。

```other
煞車皮厚度低於多少 mm 必須更換？
```

答案： 低於 **2mm**。

```other
動力系統（馬達、控制器等）標準保固期限？
```

答案： **3 年或 30,000 公里**。

```other
保固外 10 公里內道路救援基本拖吊費？夜間加成？
```

答案： **1,000 元**；夜間（22:00～06:00）**加收 50%**。
