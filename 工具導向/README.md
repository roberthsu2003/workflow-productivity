## 🛠️ 免費 AI 工具清單

### 核心對話工具（免費方案）
- **ChatGPT** (GPT-3.5) - 文字生成、對話、程式碼協助
- **Claude** (Claude 3 Haiku/Sonnet 免費額度) - 長文本處理、文件分析
- **Gemini** (Gemini Pro 免費) - Google 生態整合、多模態
- **Perplexity** (免費版) - 搜尋增強、即時資訊
- **NotebookLM** (免費) - Google 文件整合、知識庫

### 本地端工具（完全免費）
- **Ollama** - 本地運行 LLM，資料不上雲
- **Google Nano Banana Pro** - 資訊圖表、圖片生成

### 視覺設計工具（免費方案）

- **Canva** - AI 設計生成（Magic Design、Magic Write），可透過 MCP 連接 Claude Desktop

### 影片生成工具（免費方案）
- **Gemini Veo 3** - 文字/圖片生成短影片（約 8 秒）
- **Invideo AI** - 文字腳本生成完整影片（免費版有浮水印）
- **Simplified** / **Lumen5** - 文字/文章轉影片
- **VEED** / **Kapwing** - 影片編輯、自動字幕生成

### 進階工具（免費方案）
- **MCP** (Model Context Protocol) - 工具整合框架
- **RAG** (檢索增強生成) - 本地文件知識庫
- **Claude Skills** - Claude 客製化工具
