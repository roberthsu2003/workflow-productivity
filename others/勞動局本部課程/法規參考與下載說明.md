# 法規參考與下載說明（教學用）

本課程強調：**對外文字須能追溯至法令或授權依據**。使用生成式 AI 時，應先備妥**正式法規文本**（通常為 PDF），**上傳至對話或知識庫**後再下達指令，並要求模型**僅依所附文件**援引或概述，**不得**憑訓練記憶臆測條號或文義。

## 取得「最新」文本的建議做法

- **主來源**：[法務部全國法規資料庫](https://law.moj.gov.tw/)（法規隨時修正，以該庫**登載之最新版本**為準）。
- **下載或匯出 PDF 的常見方式**（介面可能微調，以網站實際按鈕為準）：
  1. 進入下列各法之「所有條文」頁面後，使用頁面上的 **「下載」** 取得檔案；或
  2. 使用 **「友善列印」**，在瀏覽器列印對話框中選 **「儲存為 PDF」**，以取得單次快照（請註明列印／下載日期以利版本管理）。

> **注意**：教學範例連結僅供快速進入該法規頁面；實際承辦業務時，仍應以當時全國法規資料庫登載內容及機關內部函釋為準。部分法規頁面會標示**條文生效狀態**或**施行日期由行政院定之**等資訊，請以頁面「沿革」與主管機關公告為準，勿僅憑舊版 PDF 撰文。

## 與 AI 協作時的簡要檢核

1. 已上傳之法規 PDF 是否為**近期自官方資料庫取得**之版本。
2. Prompt 是否明載：**僅依附件／上傳檔案**說明，不引用附件未出現之條次。
3. 產出若將對外發布，仍須經**承辦人審閱**並依機關簽核程序辦理。
