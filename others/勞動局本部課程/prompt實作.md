# 四、老師示範案例（4分鐘）

主題：回覆勞工申訴加班費未給付案件

第一步（模糊問法）：

幫我寫回覆

第二步（加入角色）：

你是一位臺北市政府勞動局承辦人， 請回覆一位勞工申訴公司未依法給付加班費的案件。

第三步（完整版本）：

你是一位臺北市政府勞動局承辦人。

案件背景： 民眾反映其任職公司長期要求加班， 但未依勞動基準法規定給付加班費。

請撰寫正式回覆公文， 需包含：

1. 申訴事項說明
2. 勞動局處理流程（如勞動檢查或調解機制）
3. 後續協助與聯絡方式 字數約300字。

觀察三次產出的差異。

# 五、學生實作任務（6分鐘）

## 任務：勞動權益宣導公告

情境： 近期發現部分企業未落實加班費與例假日規定， 需加強對企業與勞工之法規宣導。

請完成一份對外公告稿。

要求：

1. 指定角色（臺北市政府勞動局承辦人）
2. 補充完整背景
3. 指定字數（200字左右）
4. 語氣正式且清楚易懂
5. 內容需包含：合法工時原則與加班費基本說明

提示模板：

你是一位臺北市政府勞動局承辦人。

請協助撰寫一份勞動權益宣導公告。

背景說明： （填入具體情境）

請用公告格式輸出， 語氣正式且易懂， 字數約200字。
