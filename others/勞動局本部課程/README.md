### 注意事項

1. 上課對象為一般使用者無資訊背景，技術細節簡單說明即可，主要著重在實作。
2. 希望帶著同仁實作幾項公務上常應用的事項，舉例如文字生成（公文、新聞稿）、圖像影像生成（臉書圖卡、宣導廣告影像）、文字圖像整合（簡報製作)、語音轉文字與摘要（會議記錄、訪談紀錄）。
3. 除實作的幾款軟體外，煩請老師再多推薦介紹並演示幾款實用性高的 AI 軟體工具。
4. 考量安裝程序及電腦硬體限制，地端軟體屆時請老師進行示範操作即可，同仁電腦不另行安裝。
