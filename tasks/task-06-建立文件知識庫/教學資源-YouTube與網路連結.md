## 📌 快速搜尋 YouTube 的建議步驟

### 給老師的搜尋建議

1. **第一步：確定搜尋目標**
   - 想要找 NotebookLM 的使用教學？
   - 想要找 Ollama 本地 RAG 的建立方法？
   - 想要找 RAG 技術的理論基礎？

2. **第二步：組合關鍵字**
   - 中文教學：使用繁體中文關鍵字
   - 英文教學：使用英文關鍵字可以找到更多專業內容
   - 加上年份：確保找到最新的教學影片

3. **第三步：篩選結果**
   - 觀看次數：選擇觀看次數較多的影片（通常品質較好）
   - 上傳日期：選擇最近一年內的影片（工具可能會更新）
   - 影片長度：10-30 分鐘的影片適合課堂播放

4. **第四步：預先觀看**
   - 在推薦給學生前，先完整觀看一遍
   - 確認內容是否符合教學需求
   - 注意是否有廣告或不適合的內容

### 給學生的搜尋建議

1. **自主學習時可以搜尋：**
   - "NotebookLM 教學"
   - "如何建立文件知識庫"
   - "RAG 技術入門"

2. **遇到問題時可以搜尋：**
   - 具體的問題描述，例如："Ollama 安裝錯誤"
   - 加上"教學"或"tutorial"關鍵字
   - 參考多個影片，比較不同教學方式

## ⚠️ 使用 YouTube 影片的注意事項

1. **版權問題**
   - 影片為他人製作，使用時請遵守版權規定
   - 課堂使用通常屬於教育用途，但建議確認影片授權
   - 不要下載影片重製，建議直接分享連結

2. **內容審核**
   - 在課堂播放前，請先完整觀看影片
   - 確認內容適合學生年齡和程度
   - 注意是否有不當廣告或連結
   - 確認技術內容的準確性

3. **連結維護**
   - YouTube 影片可能會被刪除或設為私密
   - 建議建立播放清單並定期檢查
   - 準備備用影片或替代方案

4. **網路連線**
   - 確保課堂有穩定的網路連線
   - 可以事先下載影片（若授權允許）
   - 準備離線版本的教學內容作為備用
