# 教務處客服原始檔

## 模擬學生來信詢問

### 📩 範例一：關於【選課】的詢問

**主旨：選課系統登不進去，還有加退選的問題**

Plaintext

```other
教務處老師您好：

我是美術系二年級的學生，我想請問一下，我今天早上想要登入系統加選「基礎素描」，但是一直顯示密碼錯誤，請問我是要打哪一個密碼？

另外，因為我這學期想要超修到 28 學分，系統一直跳出說超過上限不讓我選，請問這要怎麼辦？還有加退選到底是什麼時候結束？我很怕錯過時間，再麻煩老師回覆了，謝謝。

學生 林大明 敬上
學號：112040XX
```

### 📩 範例二：關於【成績】的詢問

**主旨：詢問期末成績複查流程與預警通知**

Plaintext

```other
老師您好：

我剛剛在學生資訊系統看到我的「數位媒體設計」成績只有 55 分，但我自覺期末作業寫得很認真，平時也都有出席，懷疑是不是老師漏改了我的報告或是分數算錯。

請問如果我要申請成績複查的話，最晚要在什麼時候提出？需要準備什麼文件嗎？

另外，我爸媽昨天說收到學校寄的預警通知，是因為我期中考有科目不及格嗎？收到這個會被退學嗎？我很擔心，謝謝老師。

學生 陳小華 
學號：111020XX
```

### 📩 範例三：關於【畢業門檻】的詢問

**主旨：大四畢業門檻核對與英文門檻替代方案**

Plaintext

```other
教務處客服老師您好：

我是應屆畢業生，目前正在確認畢業學分。我想請問 128 學分是包含所有的選修嗎？如果我的系訂必修都過了，但通識還差一門，這樣可以畢業嗎？

還有關於英文門檻，我考了兩次多益都沒有達到 550 分，請問學校有什麼補救方案嗎？還是說我一定要考到過才能領證書？

最後想確認一下，如果都符合資格，大概幾月可以拿到畢業證書？離校手續要在哪裡辦？感謝老師辛苦回覆！

哲學系四年級 王小明 敬上
學號：109010XX
```
