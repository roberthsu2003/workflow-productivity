# 週報整理助手原始檔

## 檔案說明

| 檔案 | 用途 |
|------|------|
| [週報範本.docx](./週報範本.docx) | 週報應呈現的**章節與欄位**（標題層級、表格或區塊可依此對齊）。 |
| [當週工作記錄_週一.docx](./當週工作記錄_週一.docx)～[當週工作記錄_週五.docx](./當週工作記錄_週五.docx) | 當週**每日**工作筆記（口語、條列皆可），**每週整理時**再提供。 |

**使用方式**：

- **Claude Projects**：建議將 **`週報範本.docx` 預先上傳至專案知識**（或釘選），之後各週沿用；**週一～週五**工作記錄則在該週對話中上傳或貼上即可。
- **Gem／GPT**：可將範本與當週日記一併上傳至知識庫／相關資訊，或將內容複製貼上對話。
- 若介面無法直接讀取 `.docx`，請改貼文字或匯出為 `.txt`／`.md` 再上傳。

綜合練習的建議 Prompt 請見上層 [儲存與重複使用 AI 提示詞 README](../README.md) 的「🚀 綜合練習」。
