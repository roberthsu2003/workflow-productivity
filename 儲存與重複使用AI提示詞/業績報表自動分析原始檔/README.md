# 業績報表自動分析 — 教學用 Excel 範例

本資料夾內含三份 **.xlsx**，供上層單元 [儲存與重複使用 AI 提示詞](../README.md) 中「**✏️ 小練習（Claude Projects）：業績報表自動分析**」使用。內容為**教學虛構**（姓名、金額等請視為範例），勿作為真實商業或個資依據。

## 檔案一覽

| 檔案 | 說明 |
|------|------|
| [銷售業績範例_簡單版.xlsx](./銷售業績範例_簡單版.xlsx) | 結構較單純、欄位清楚，適合**第一次**跑通「上傳 → 四區塊分析輸出」全流程。 |
| [銷售業績範例_中等版.xlsx](./銷售業績範例_中等版.xlsx) | 複雜度介於簡單與進階之間，適合練習**排名、趨勢摘要、異常偵測**（仍建議搭配同一套 ROSES）。 |
| [銷售業績範例_進階版.xlsx](./銷售業績範例_進階版.xlsx) | 含**多個工作表**等較進階情境，適合練習**辨識工作表／欄位**與**邊界情況**（缺欄、跨表對照時 AI 是否先說明結構再分析）。 |

## 使用方式

1. 在 Claude 專案對話中**上傳**任一份 `.xlsx`（或依教學先放入專案 **Files** 再在對話引用）。
2. 專案 **Instructions** 請貼上主 README 內「建議 Prompt」摺疊區的 **ROSES 全文**。
3. 確認已啟用 **xlsx**（或介面同等試算表讀寫能力），否則請改以匯出 CSV 等方式並自行調整提示詞說明。
