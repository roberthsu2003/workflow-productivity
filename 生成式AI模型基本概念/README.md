# 生成式AI模型基本概念

## 為什麼 ChatGPT 的功能愈來愈多？

### 核心概念：從「模型」到「應用程式」

**ChatGPT 的本質演進：**

| 階段 | 本質 | 功能 |
|------|------|------|
| 最初 | **語言模型** | 輸入文字 → 輸出文字（單純的生成） |
| 現在 | **應用程式** | 文字、圖片、語音、搜尋、檔案處理、外掛程式…… |

### 關鍵理解

1. **底層能力不變**
   - 大型語言模型（LLM）的核心能力始終是：根據輸入產生合理的文字輸出
   - 這就像「引擎」——引擎本身只負責運轉，不決定車子能做什麼

2. **應用程式層的擴充**
   - ChatGPT 是一個**應用程式**，它把語言模型「包」在裡面
   - 應用程式可以不斷加入新功能，例如：
     - 上傳圖片並解讀
     - 語音輸入與輸出
     - 搜尋即時資訊
     - 讀取、分析檔案
     - 串接第三方工具（外掛程式）

3. **比喻說明**
   - **模型** = 廚師（會做菜）
   - **應用程式** = 餐廳（有菜單、點餐系統、外送服務、會員制度……）
   - 餐廳可以不斷擴充服務，但廚師的核心技能還是「做菜」

### 總結

> ChatGPT 功能變多，是因為它是一個**應用程式**。應用程式可以在內部持續整合、擴充各種功能，而底層的語言模型則提供最核心的「理解與生成」能力。
