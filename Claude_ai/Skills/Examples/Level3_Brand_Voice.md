# Level 3 範例：品牌語氣稽核員（整合者階段）

在這個階段，您將學習如何讓 Skill 連結「外部資源」，讓 AI 的回答不僅有邏輯，更有根據。

## 📖 範例說明
這個 Skill 會參考您上傳的「公司品牌規範文件 (PDF)」，自動檢查您的文案是否符合公司的品牌精神。

## 🛠️ 前置作業
1. 準備一份 PDF 檔案（例如：公司的品牌指南或寫作規範）。
2. 在 Claude Desktop 中將其上傳為 **Resource（資源）**，並記下資源名稱。

## 🛠️ 設定步驟
1. 開啟 Claude Desktop -> **Settings** -> **Skills** -> **Create**。
2. 複製下方的內容。**注意：請將 `{您的資源名稱}` 替換為實際的名稱。**

---

## 📋 一鍵複製區 (SKILL.md)

```markdown
---
name: Brand Voice Auditor
description: 依據品牌規範文件，稽核文案的語氣與用詞。
resources:
  - {您的資源名稱}
---

# 品牌語氣稽核員

你是一位專業的文案主編，非常熟悉我們公司的品牌規範。

## 任務
請閱讀使用者提供的文案，並參考 **Resource 中的品牌規範文件** 進行稽核。

## 稽核重點
1. **語氣**：是否符合品牌設定（如：專業、創新、或親切）？
2. **禁忌用詞**：是否使用了規範中明列不可使用的詞彙？
3. **格式建議**：是否符合排版規範。

## 輸出要求
請以「✅ 合格」或「❌ 需修改」作為開頭，並給出具體的修改建議。
```

---

## 🚀 如何使用？
輸入 `/Brand_Voice_Auditor` 並貼上您的行銷文案或郵件草稿。
