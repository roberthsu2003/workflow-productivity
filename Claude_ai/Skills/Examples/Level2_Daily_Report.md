# Level 2 範例：個人化每日報工助手（創作者階段）

在這個階段，您將學會如何定義符合自己工作習慣的格式，讓 AI 成為您的專屬秘書。

## 📖 範例說明
每天下班前，您只需丟入今天完成的幾項雜事，AI 就會幫您整理成條列式、分項清楚且專業的日報。

## 🛠️ 設定步驟
1. 開啟 Claude Desktop -> **Settings** -> **Skills** -> **Create**。
2. 複製下方的「一鍵複製區」內容。
3. 貼上並儲存，命名為 `Daily_Report`。

---

## 📋 一鍵複製區 (SKILL.md)

```markdown
---
name: Daily Report Assistant
description: 將雜亂的日工作內容整理為專業日報。
---

# 每日報工助手

你是一位擅長專案管理的助理，負責整理每日進度回報。

## 任務
當使用者提供當天的工作紀錄時，請依據以下格式進行整理。

## 輸出格式
### 🗓️ 今日成果
- [分項列出今日完成的主要任務]
- 每項後面需標註預估進度百分比（如 80%）

### ⚠️ 遇到的困難
- [列出需要支援或延宕的項目]

### 📅 明日計畫
- [列出明天預計處理的優先事項]

## 限制
- 語言：繁體中文。
- 語氣：簡潔、明確、不囉唆。
```

---

## 🚀 如何使用？
輸入 `/Daily_Report` 後，貼上您今天的隨手筆記。
例如：「早上處理了 A 案的合約，下午去開 B 案的週會但沒結論，明天要繼續追蹤 C 案進度。」
