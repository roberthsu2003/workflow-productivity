# Settings（設定）：解鎖 Claude 的隱藏能力

要發揮 Claude 的完整威力，必須學會管理 **Settings**。這裡不僅是調整個人資料的地方，更是開啟「電腦操作」、「數據分析」與「自動化」的關鍵開關。

---

## 🛠️ 桌面應用程式設定 (Desktop App - General)

在 Claude Desktop 左下角點擊頭像 -> **Settings** -> **General**。

### 1. 快捷啟動 (Quick access shortcut)
- **功能**：設定一組快捷鍵（如雙擊 `Option` 鍵），讓您可以隨時在螢幕任何地方叫出 Claude。
- **開啟建議**：**強烈建議開啟**。這是提升工作流效率的關鍵，不需切換視窗即可提問。

### 2. 瀏覽器連動 (Browser Use)
- **功能**：允許 Claude 在 Chrome 中自動執行操作（需安裝 Chrome 擴充功能）。
- **開啟建議**：如果您需要 Claude 幫您搜尋網頁、摘要當前頁面，請務必開啟。

### 3. 電腦操作模式 (Computer use - Beta)
- **功能**：讓 Claude 可以「看見」您的螢幕、移動滑鼠並點擊按鈕。
- **權限需求**：在 macOS 上需授權「螢幕錄製」與「輔助使用」。
- **備註**：通常僅限 **Pro** 或 **Max** 方案。

---

## 🚀 核心能力開關 (Capabilities / Feature Preview)

這些功能可能位於 **Settings -> Capabilities** 或 **Feature Preview** 頁面。

| 功能名稱 | 用途描述 | 什麼時候要開？ |
| :--- | :--- | :--- |
| **Artifacts** | 產生側邊獨立視窗顯示程式碼、文件、圖表。 | **必開**。開發、寫作、製作流程圖時最實用。 |
| **Analysis Tool** | 內建 Python 執行環境，可處理 Excel/CSV 並產出圖表。 | 需要進行數據統計、精密運算時開啟。 |

---

## 📋 功能對應清單：我要如何開啟...？

- **我要串接 Google Drive 或 GitHub**：
    - 前往 **Settings -> Connectors**。
- **我要手動設定本地端的 MCP Server**：
    - 前往 **Settings -> Developer**（修改 JSON 設定檔）。
- **我要管理訂閱方案或帳單**：
    - 前往 **Settings -> Billing**。

---

> **💡 小提醒**：部分功能（如 Computer use）在開啟後，可能還需要您在對話中切換到特定的模式（如 Cowork 模式）才能真正觸發。
